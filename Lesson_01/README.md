# Lesson 1: HTML Fundamentals

This lesson introduces the core structure of an HTML page and the most common HTML tags.

## Learning Objectives

By the end of this lesson, students should be able to:

- Create a valid HTML document structure
- Use headings, paragraphs, and lists correctly
- Add links and images to a page
- Use basic semantic tags for page structure

## Lesson Files

- `project/index.html` - starter example with basic HTML tags
- `PRACTICE.md` - practice tasks for this lesson

## Topics Covered

1. HTML document structure

- `<!DOCTYPE html>`
- `<html>`, `<head>`, `<body>`
- `<meta charset>` and viewport tag

1. Text and content tags

- Headings: `<h1>` to `<h6>`
- Paragraph: `<p>`
- Lists: `<ul>`, `<ol>`, `<li>`

1. Links and images

- Link: `<a href="...">`
- Image: `<img src="..." alt="...">`

1. Semantic structure

- `<header>`, `<main>`, `<section>`, `<footer>`
