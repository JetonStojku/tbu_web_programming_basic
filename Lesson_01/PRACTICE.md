# Lesson 1 Practice

Use `project/index.html` for all tasks in this lesson.

## Practice Tasks

1. Change the page title (`<title>`) to your own website title.
2. Add a new `About Me` section with 3-4 sentences.
3. Add an ordered list with your top 3 favorite websites.
4. Replace the current image URL with another image URL.
5. Add one external learning link (for example W3Schools HTML page).
6. Add a footer with your name and the year.

## Expected Output Checklist

- Page has one clear `<h1>`.
- Sections use semantic tags (`header`, `main`, `section`, `footer`).
- At least one working external link exists.
- At least one image with meaningful `alt` text exists.
- Ordered list is visible and correctly structured.

## Extra Challenge (Optional)

1. Add a small top navigation with internal links (`#about`, `#favorites`, `#contact`).
2. Add a second image and compare both with captions.
3. Add one `article` block inside a section.

## Self-Review Questions

- Is heading order logical (`h1` then `h2`)?
- Are all tags properly nested and closed?
- Would another student understand the page structure quickly?
