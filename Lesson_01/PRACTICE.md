# Lesson 1 Practice

Use `project/index.html` for all tasks in this file.

## Practice Tasks

1. Open `project/index.html` and change the page title.
2. Add one more section with your short biography.
3. Add an ordered list with your top 3 favorite websites.
4. Replace the placeholder image with another image URL.
5. Add one external link to a W3Schools page.

## Extra Challenge (Optional)

1. Add a second image with a meaningful `alt` text.
2. Create a small navigation list at the top with links to sections on the same page.
3. Use at least one semantic tag you did not use before.
