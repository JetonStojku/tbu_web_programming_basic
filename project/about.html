About This Demo

This website is a static demonstration project for Bootstrap 5.

All movies, comments, and ratings are static content for educational purposes only.

  • Bootstrap components: navbar, cards, badges, buttons, list groups.
  • Bootstrap layout: container, rows, and columns.
  • Custom CSS: colors, card style, and cover sizing in `styles/style.css`.
